About Project

LOOW application is built using azure services, front end UI is built using html,css and backend data stored in cosmodb.

Azure function:-

  • Httptrigger:- which used to save event data in cosmodb.

  • Timetrigger:- which send mail communication to Event admin about user in a timely manner.

  • Azure cosmodb(Backend source):-User Data stored in cosmodb.

  • Sendgrid twilio:-Mail communication sent to Event admin in a timely manner by Azure function time trigger.

  • Github:-Application hosted in github.
